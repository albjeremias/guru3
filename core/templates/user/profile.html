How to delete your account

If you want to delete your account please change your First and Last name to Please Delete and send a message to poc@eventphone.de with the username and email address of your account.

We will delete all your data and send you a confirmation.

User API key

This will generate a new API key for your user. The API key can be sent in an ApiKey HTTP header to authenticate with your user as an alternative to login and session cookie. For security reasons (API keys are often used as machine credentials) API key authentication is only enabled for the /api/ sub space of urls.

The available API is or will in a continuing fashion be documented in our documentation Wiki. If you have use-cases that are currently not reflected in the API, feel free to contact us via our support system.

Please note that this action will immediately invalidate the old API key. Moreover, the API key will only be shown once here. If you lose it or want to reset it, just generate a new one.
